# 워드 클라우드 (Word Cloud) 작동 방법
## 전체 처리 과정
### 1. 단어 추출
텍스트 전처리 진행
* 정규화
    * 소문자 변환
    * 불용어(stopwords) 제거: 조사, 접속사 등 의미 없는 단어 제거
    * 특수문자 및 숫자 제거
* 토큰화: 문장을 단어 단위로 분리


### 2. 단어 중요도 계산
* 빈도수 기반 방법
    > '많이 등장한 단어일 수록 중요하다.'
    * 중요도 = 단어 등장 빈도
* TF-IDF 기반 방법
    > '어떤 문서에서 자주 등장하지만, 다른 문서에서는 잘 등장하지 않는 단어가 그 문서를 대표한다.'
    
    빈도수 기반 방법보다 더 정교한 분석에 사용됨
    * **TF(Term Frequency)**: 문서 내 등장 빈도
    * **IDF(Inverse Document Frequency)**: 여러 문서에서의 희소성
    * 중요도 = TF x IDF
* WordCloud 라이브러리는 기본적으로 빈도 기반 방법 채택하고 있음
    * `WordCloud.generate()` 대신 `WordCloud.generate_from_frequencies()`를 사용하면 TF-IDF를 계산한 결과를 빈도처럼 넘겨서 적용 가능


### 3. 단어 배치 알고리즘
단어가 겹치지 않도록 배치해야 하며, 일반적으로 다음과 같은 방식이 사용됨
* 단어의 크기는 중요도에 비례
* 가장 큰 단어부터 배치
* 단어별 배치 루프 (max_words번 반복)
    * 회전 각도 결정
    * 단어를 그렸을 때의 실제 박스 크기 측정
    * 후보 자리 탐색 루프
        * 중심에서 바깥으로 퍼지는 나선형 경로로 좌표 생성
        * 좌표에 박스를 배치했을 때 픽셀 단위 충돌 검사
        * 충돌하지 않을 때까지 다음 좌표 탐색
