# 리뷰
### 1. SQL 학습
* SQL example들을 sqlite 환경에서 직접 실행하며 동작을 확인함
* 결과를 직접 확인하며 쿼리 결과와 테이블 구조 변화를 비교함
* sqlite가 지원하지 않는 기능은 다른 문법으로 대체할 수 있는지 찾아보고 직접 적용해보며 원리를 이해함

### 2. ETL 프로젝트
* Region 매칭 로직 검토
    * 국가 -> Region 매칭 과정에서 India가 Africa로 잘못 매칭된 사례를 발견함
        * 매칭 로직으로 가장 유사도가 높은 국가명을 선택하는 방법은 적합하지 않음을 깨달음
        * 국가명이 정확하게 일치할 때만 Region을 반환하는 방법으로 수정
    * Region 매칭에 시간이 오래 걸리는 것을 확인
        * 국가마다 매번 API 요청을 보내는 방법에서 처음 한번만 모든 국가에 대한 Region을 딕셔너리로 만든 후 딕셔너리를 참조하여 매칭하는 방법으로 변경하여 실행 시간 대폭 단축
* 과거 데이터 관리 관점의 구조 검토
    * 과거 데이터가 필요해질 경우를 가정하여 테이블에 갱신 시점 컬럼이 필요함을 정리함
