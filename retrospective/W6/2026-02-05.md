# 리뷰
## 아이디어 실현 가능성 검토 (오전)
### 법/제도 검토
* 일반 식당 주차장을 화물차 공동주차장으로 활용할 수 있는지 관련 법과 지자체 규정을 직접 조사함
    * 화물차 차고지 등록, 공동주차장 요건, 지목 조건 등 여러 법이 얽혀 있어 챗지피티한테 찾아달라고 해도 잘 못 찾아줬음

* 시간이 너무 오래 걸리는 것 같아서 방식 전환 → 지자체 직접 문의
    * 광주시에 전화하여 관련 조건을 직접 문의함

    * 30분 이상 혼자 찾던 내용이 통화 몇 분 만에 정리됨

    * 추가로 궁금했던 조건까지 한 번에 확인할 수 있었음

    * 혼자 검색하는 것보다 웬만하면 직접 문의가 훨씬 빠르겠다고 생각함

### 식당 주인 인터뷰
* 과거 식당 운영 경험이 있는 친오빠에게 전화로 물어봄
    * 영업 종료 후 야간 시간에 화물차 1대 주차 → 월 15만원 수익
    * 반응: 영업 시간 외에만 주차한다는 조건을 계약서에 명시하면 부수입이 생기는 거라 금액이 작아도 무조건 한다는 반응
    * 단, 기사 자가용을 낮 시간까지 주차하는 것은 손님 차량을 못 받아서 불가
* 타겟 조건을 수정
    * 자가용 없이 출퇴근 가능한 기사로 범위 좁힘
    

## 공급자 수요 확인 (오후)
* 광주시 기준 화물차 공동주차장 조건에 맞는 식당 탐색
    * 생각보다 많은 식당의 지목이 주차장으로 가능한 '대지'였고, 주거지역과 인접한 곳도 엄청 많았음
    * 주거지역별로 1곳만 확보해도 공급 측면은 충분히 가능성 있다고 판단함
* 실제 전화 인터뷰 진행
    * 시흥시, 화성시 식당 대상으로 전화 수요조사 시도
    * 대부분 사장님 부재로 통화 실패
    * 일부 연결된 경우, 스타트업 수요조사라고 하니까 바로 끊어버리심
    * 친오빠의 긍정적인 반응으로 수요조사 결과를 기대했는데, 모르는 사람을 대상으로 하는 수요조사는 생각보다 훨씬 벽이 높았음
    * 대본을 너무 솔직하게 짠 것 같아서 내일은 "기대 수익 60만원인데 혹시 관심있으신가요?" 이런 방식으로 혹 할 만한 요소를 먼저 제시하면서 인터뷰 해보기로 함

# 학습 기계
## 잘 지키고 있음
1. 매주 금요일 퇴근 전 10분 동안 일주일 동안 쓴 회고를 다시 읽고 리뷰하기
    - 잘 못 지킨 습관은 실천 방법 수정하기
2. 매주 금요일 피드백 받은 내용 최소 1가지 적용해보기
3. 설계에 대한 의사결정을 하기 전 usecase 최소 2가지 생각하기 

## 노력 필요
1. 모든 작업 최대 30분씩 타임박싱 하고, 연장은 최대 1번만 하기 (작업 = 해결 할 작은 문제 1개 e.g., 도커 파일 작성)
2. 데드라인보다 하루 더 일찍 끝내는 것을 목표로 스케줄링 하기
3. 미션 1개 해결 이후 10분 동안 해당 미션 리뷰하기

## 하면 좋음
1. 집중력이 깨지면 1차 스트레칭, 그래도 집중이 안되면 양치해보기
2. 팀원들과 팀미션 시간에 개인 미션 구현 방식을 3분씩 설명하고 리뷰하는 시간 갖기
