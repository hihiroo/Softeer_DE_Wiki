# 리뷰
### W2M2 ~ M4
* Multiprocessing 모듈의 기본 사용법 찾아보고 미션 코드 작성
* URL이나 data path 같은 부분은 yaml 파일에 저장하고 불러올 수 있도록 코드 수정
* W2M3 진행 중 같은 코드임에도 결과가 제대로 나왔다가 안나왔다가 하는 현상 발견
    * chatgpt에 물어본 결과 `empty()`가 올바른 결과를 반환하지 않을 수도 있음을 알게 됨
    * `empty()`를 대체할 수 있는 방법 탐색 결과 try exception을 활용하는 방법이 있음을 알게 됨
        * 에러가 발생할 때까지 큐에서 `get()`
    * try exception 구조로 변경했을 때 무한루프 발생
        * chatgpt를 통해 `get()`은 아이템이 있을 때까지 blocking 되기 때문에 `get_nowait()`을 써야 함을 알게 됨
